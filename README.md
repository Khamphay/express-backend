#Initialize a TypeScript

```
1. npm init -y
2. npm install typescript ts-node @types/node --save-dev
3. npx tsc --init
4. npm install prisma --save-dev
5. npx prisma init --datasource-provider mysql
6. npm install @prisma/client
7. npx prisma migrate dev --name init //init, migrate and generate models
8. npx prisma generate //Re-generate models
9. npx prisma db push //Update database for sync with schema
9. npx prisma db pull //Update schema sync with database
```
